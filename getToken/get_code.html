    Авторизация для работы с API Яндекс Директ с помощью R клиента ryandexdirect

    Ваш Код для получения токена:

    Введите значение данного токена в R консоль в качестве ответа на запрос "Enter authorize code:", после чего в R будет создан объект содержащий токен доступа к API Яндекс Директ и вы сможете получить данные с помощью функций доступных в пакете ryandexdirect.

    Подробное описание всех доступных функций пакета можно найти тут.
